## 9. 附录

### 9.1 任务状态定义

| 状态 | 符号 | 定义 |
|------|------|------|
| 未开始 | ⚪ | 任务尚未开始 |
| 进行中 | 🟡 | 任务已开始，未完成 |
| 已完成 | ✅ | 任务完成，验收通过 |
| 延期 | 🔴 | 任务未按计划完成 |
| 阻塞 | 🛑 | 任务因依赖无法继续 |

### 9.2 进度计算方式

```
整体进度 = Σ(任务权重 × 任务进度) / Σ任务权重

任务权重:
- 关键路径任务：2x
- 非关键路径任务：1x
```
